## Mini Project: Civic Voice Interface (CVI)

The `civic_voice/` folder is a complete mini-project built with Flutter. It demonstrates a voice-first civic assistance application intended for use in an Indian context.

### Key features

- Voice-enabled conversational assistance
- Multilingual support (English, Hindi, Marathi, Tamil)
- Government service guidance for Aadhaar, PAN, Passport, Driving License, and more
- Text-to-Speech and Speech-to-Text integration
- Authentication flows and guest access
- Dashboard analytics and usage tracking

### Notable files and folders

- `civic_voice/lib/` - Flutter app source code
- `civic_voice/assets/` - app assets, images, icons, and Lottie animations
- `civic_voice/README.md` - detailed project description and technical overview
- `civic_voice/android/`, `civic_voice/ios/`, `civic_voice/web/`, etc. - platform-specific build files

## How to Use

### Opening assignment files

1. Navigate to the `assignments/` folder.
2. Open the desired `.pdf` or `.docx` file.

### Running the mini-project

1. Open the `civic_voice/` folder in a Flutter-capable IDE (Visual Studio Code or Android Studio).
2. Ensure Flutter SDK and required tools are installed.
3. Run `flutter pub get` from the `civic_voice/` directory.
4. Launch the app on an emulator or physical device.
